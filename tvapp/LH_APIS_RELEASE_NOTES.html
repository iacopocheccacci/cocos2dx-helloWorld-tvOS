After Update Compilation Help

Because of changes in the Cocos2d-X 3.3 library files & code structure, user action is required to make an old project compile.
1. Go to AppController.mm and change:
#import "CCEAGLView.h"
to
#import "platform/ios/CCEAGLView-ios.h"
2. Go to RootViewController.mm and change:
#import "CCEAGLView.h"
to
#import "platform/ios/CCEAGLView-ios.h"
3. Go to AppController.mm and change:
cocos2d::GLView *glview = cocos2d::GLView::createWithEAGLView(eaglView);
to
cocos2d::GLView *glview = cocos2d::GLViewImpl::createWithEAGLView(eaglView);
4. Go to AppDelegate.cpp and change:
glview = GLView::create("My Game");
to
glview = GLViewImpl::create("My Game");
5. Select all your project targets and under "General" tab find "Linked Frameworks and Libraries" and remove the following libraries if present:

libbox2d iOS.a
libchipmunk iOS.a
libcocos2dx-extensions iOS.a
libCocos2dDenshion iOS.a
